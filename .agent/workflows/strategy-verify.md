# 策略驗證工作流 (Strategy Verification Workflow)

此工作流用於在**收盤後**驗證當日 Session 中關注股票的表現，並確認早盤/盤中的分析策略是否有效。

## 1. 準備工作
- [ ] 讀取當前的 Session 檔案 (例如 `sessions/session_YYYY-MM-DD.md`)
- [ ] 提取該 Session 中所有已分析的標的 Ticker (例如 2330.TW, 1513.TW, POET)
- [ ] 提取每個標的的「建議買進價」、「風險防線」與「目標價」

## 2. 數據回測 (Data Backtesting)
對每一個提取的標的，執行以下檢查：

### A. 價格驗證
- [ ] 獲取今日 **收盤價 (Close)**, **最高價 (High)**, **最低價 (Low)**
- [ ] **是否觸發進場**?
    - 檢查今日 `Low` 是否低於或觸及 `建議買進價`
    - 若是，標記為 `[已進場]`，並記錄假設進場價格
- [ ] **是否觸發停損**?
    - 檢查今日 `Low` 是否低於 `風險防線`
    - 若是，標記為 `[停損觸發 ❌]`
- [ ] **是否觸發獲利**?
    - 檢查今日 `High` 是否高於 `目標價`
    - 若是，標記為 `[達標 ✅]`

### B. 策略有效性評分
- [ ] **準確度判定**:
    - 若觸發進場且收盤價 > 進場價 -> **勝 (Win)**
    - 若觸發進場且收盤價 < 進場價 -> **負 (Loss)**
    - 若未觸發進場 -> **觀望 (Wait)**

## 3. 更新 Session 檔案
在 `sessions/session_YYYY-MM-DD.md` 的底部添加「策略驗證報告」章節：

```markdown
## 📝 盤後驗收單：今天策略有用嗎？ (小白驗證版)
**驗收時間**: YYYY-MM-DD HH:MM (台股收盤囉！來看看今天的成績單)

| 股票是誰 | 我們預計哪裡買 | 今天最低跌到哪 | 收盤價 | 結果 | 小白白話解讀 |
|---|---|---|---|---|---|
| **2330.TW (台積電)** | 1700-1750 | 1740 (中了!) | 1755 | ✅ 賺錢啦 | 股價真的跌下來讓我們上車，收盤已經還是賺的！ |
| **3466.TWO (德晉)** | 38.0-39.5 | 39.10 (中了!) | 39.40 | ✅ 小賺 | 跌下來剛好接到，收盤守住，安全！ |
| 1503.TW (士電) | 205-210 | 215 (太高了) | 218.5 | ⏸️ 看戲 | 今天太強了，根本不下來給我們買。 |
| POET (POET Tech) | 7.5-8.5 | - | - | ⏳ 還沒開 | 美股晚上 22:30 才開盤，睡醒再來看。 |

### 📊 今天的成績單總結

- **我們總共盯了幾支**: X 支台股。
- **真正買到的**: Y 支。
- **勝率**: **Z%** (買到的目前都賺錢)。
- **沒買到的**: W 支，因為它們今天太強了，沒跌下來給我們便宜買。

### 💡 給小白的盤後悄悄話
1.  **好股票跌下來就是機會**: 你看 [成功例子]，敢在跌的時候買，收盤就開心了，這就是「逢低佈局」的甜頭。
2.  **太強買不到怎麼辦？**: 今天的 [強勢股] 都飛太快，我們設的買點太保守（想買太便宜）。沒買到沒關係，**「寧可少賺，不要賠錢」** 是新手的最高準則！我們不追高，才不會住在套房。
3.  **進階招式 (給想學多一點的人)**：
    - 如果股票真的強到都不跌怎麼辦？有一種叫 **「5日線買法」**，就是看它過去5天的平均價格，只要股價碰到這條線沒跌破，就可以試試看。
    - 還有一種叫 **「突破買法」**，就是等它衝過之前的最高點（天花板）時，直接跟著衝進去。但這兩個比較危險，心臟要大顆一點，我們先把基本功練好再來學！
4.  **明天看什麼？**: [提醒美股開盤或明天觀察重點]。
```

## 4. 執行指令
- 詢問用戶是否要針對驗證結果失敗的標的進行 `/deep-dive` 重新檢視。
